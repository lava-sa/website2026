# SEO Intentional Exceptions

This file tracks pages where exact string matching between URL slug, breadcrumb leaf, metadata title, and H1 is intentionally relaxed for UX/conversion copy.

## Intentional Exceptions

- `/checkout/success`
  - **Reason:** Different H1 copy for EFT vs card flow improves clarity after payment outcome.
- `/checkout/cancel`
  - **Reason:** Conversational H1 ("Your payment was cancelled") is clearer than noun-style title.
- `/rewards`
  - **Reason:** Hero H1 is conversion-led headline while title/breadcrumb remain navigational.
- `/about`
  - **Reason:** Brand-story headline prioritizes emotional positioning over structural keyword repetition.
- `/about/lasting-quality`
  - **Reason:** Story-first H1 retained for readability; metadata includes stronger SEO keyword phrase.

## Note

For all other indexable pages, we aim for close topical alignment across:

1. URL slug
2. Metadata title
3. H1
4. Breadcrumb leaf
